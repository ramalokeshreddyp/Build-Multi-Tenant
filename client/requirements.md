## Packages
framer-motion | Smooth page transitions and sophisticated UI animations
lucide-react | High quality icons for the interface
date-fns | Formatting dates for tasks and projects
clsx | Utility for conditional class names (often used with tailwind-merge)
tailwind-merge | Utility for merging tailwind classes safely

## Notes
Auth implementation expects JWT token in localStorage or httpOnly cookie.
The API routes are defined in shared/routes.ts.
The schema is defined in shared/schema.ts.
Design aesthetic: Clean, modern SaaS with Inter/Outfit font pairing, subtle shadows, and crisp borders.
